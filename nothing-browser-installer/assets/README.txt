ASSETS FOLDER
=============

Drop your files here:

assets/icons/logo.ico
  - Your Nothing Browser logo converted to .ico
  - Convert at: https://convertico.com
  - Recommended size: 256x256

assets/installer-banner.bmp
  - Sidebar image shown on Welcome + Finish pages
  - Size: 164 x 314 pixels
  - Format: BMP (24-bit)
  - Optional — delete WizardImageFile line in .iss if you skip this

assets/installer-icon.bmp
  - Small logo shown top-right on inner pages
  - Size: 55 x 58 pixels
  - Format: BMP (24-bit)
  - Optional — delete WizardSmallImageFile line in .iss if you skip this

output/
  - Compiled nothing-browser-setup.exe will appear here after you hit compile
