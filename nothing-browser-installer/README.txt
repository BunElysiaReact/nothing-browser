NOTHING BROWSER — WINDOWS INSTALLER
=====================================

REQUIREMENTS
------------
- Inno Setup 6 (https://jrsoftware.org/isdl.php)
- Inno Script Studio (https://www.kymoto.org/products/inno-script-studio)

HOW TO USE
----------
1. Drop your logo.ico into assets/icons/
2. Open nothing-browser-setup.iss in Inno Script Studio
3. Press F9 to compile
4. Find your setup.exe in the output/ folder

ASSETS (optional but recommended)
----------------------------------
See assets/README.txt for image sizes and formats.

NOTES
-----
- The installer downloads the latest release from GitHub at install time
- User needs internet connection during installation
- Tested on Windows 10 and Windows 11
